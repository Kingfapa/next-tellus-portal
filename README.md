# next-tellus-portal

# Pre-requisites
- Install [Node.js](https://nodejs.org/en/) version >=16.10
- [Yarn](https://yarnpkg.com/getting-started/install)
- VS Code (recommended)
- [git](https://git-scm.com/)

# Getting started
- Install dependencies
```
yarn install
```
- Build and run the project
```
yarn dev
```
  Navigate to `http://localhost:3000`
